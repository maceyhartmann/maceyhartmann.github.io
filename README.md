# maceyhartmann.github.io

## General Information 

This repository is for maceyhartmann.github.io . Maceyhartmann.github.io is my personal website that I made using Quarto.

This repository belongs to me, Macey Hartmann, and I am the sole contributor.

## Data and File Information

The docs folder includes the rendered HTML output files generated by Quarto and this is what gets deployed to GitHub Pages. 

The media folder includes images used for the about me page, the landing page, and the favicon. 

The project_posts folder contains individual .qmd post files for each entry on the Projects page and a media and files folders for the media and files used in the projects posts.

The scicomm_posts folder includes individual .qmd post files for each entry on the Science Communication page and a media and files folders for the media and files used in the projects posts.
